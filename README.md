# Vrinda-Store-Data-Analysis-and-Report
It's a Data Analysis projects using 'Excel' which shows customer behavior towards products and sales shows valuable business insights  also giving solution for business development.

This data analysis project is made only with 'Excel' with step by step process such as :-
Step 1 :-Data exploring(checking dataset properly).
Step 2 :-Data cleaning(missing value handling, checking misspelling, removing redundand or duplicated data, correcting every data.).
Step 3 :-Data processing.
Step 4 :-Data analysis using pivot tables to get sales/business insights,answers of queries in table format then showing table data in charts or graphs format.
Step 5 :-showing those charts in a 'dashboard' where all those charts and slicer gets included.
Step 6 :-showing main insights in written format.
step 7 :-Also giving solution for business development.
